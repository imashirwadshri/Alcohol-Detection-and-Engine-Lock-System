# 🚗 Alcohol Detection & Engine Lock System

### Arduino | MQ-3 Sensor | GPS | GSM

---

## 📌 Project Overview

This project is designed to prevent drunk driving using an MQ-3 Alcohol Sensor.

If alcohol is detected above the safe limit:

- 🚫 Engine automatically stops
- 📍 GPS fetches the current location
- 📲 GSM sends an SMS alert
- 🚨 Buzzer starts ringing

---

## 🛠 Components Used

- Arduino UNO
- MQ-3 Alcohol Sensor
- GPS Module
- GSM Module
- Relay Module
- Buzzer
- LEDs

---

## 🚀 Features

- Alcohol Detection
- Engine Lock
- SMS Alert
- Live GPS Tracking
- Low Cost
- Easy to Build
